HTML & CSS

HTML

HTML stands for Hyper Text Markup Link. HTML is used to make websites and to code. HTML came from Tim Berners-Lee who developed HTML at CERN using SGML. HTML is written in code.

CSS

CSS stands for Cascading Style Sheet. CSS is used to add style features to your HTML and make it more appealing to the eye. CSS came from html. CSS is written in the style part and it is where all of the style inputs go.

TAGS

Tags are what make your HTML work. Some tags are, style, h1, h3, color, and background. You would just add <> to the sides of the word(s).

ATTRIBUTES

Attributes are what you add to identify a part of your html. you add attributes by puting <> then whatever attribute inside. some attributes are, class, alt, id, href, and random letters put together.

SOME USEFUL TAGS
Some tags that get used alot are, style, background, and color. Style is used to style the parts on your page. Background is used to add backgrounds to the page titles and anything else. Color is used to color text. Propertys for background are images or colors. Propearts for color are the main golors and any rgb numbers.
